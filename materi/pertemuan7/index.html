Operator Perbandingan

  • Operasi perbandingan adalah operasi untuk membandingkan dua atau lebih data
  • Layaknya perbandingan pada umumnya, operasi ini mencari mana yang benar dan mana yang salah
  • Maka dari itu operasi perbandingan akan menghasilkan nilai boolean berupa true atau false
  • Nilai true akan terjadi apabila operasinya benar
  • Nilai false akan terjadi apabila operasinya salah
    • Operator Nama Deskripsi Contoh Hasil
    == Sama dengan Memeriksa apakah dua nilai sama (tanpa memeriksa tipe data). 5 == '5' true
    === Identik Memeriksa apakah dua nilai sama dan tipe datanya juga sama. 5 === '5' false
    != Tidak sama dengan Memeriksa apakah dua nilai tidak sama (tanpa memeriksa tipe data). 5 != '5' false
    !== Tidak identik Memeriksa apakah dua nilai tidak sama atau tipe datanya berbeda. 5 !== '5' true
    > Lebih besar Memeriksa apakah nilai di sebelah kiri lebih besar daripada nilai di sebelah kanan. 5 > 3 true
    >= Lebih besar atau sama dengan Memeriksa apakah nilai di sebelah kiri lebih besar atau sama dengan nilai di sebelah kanan. 5 >= 3 true
    < Lebih kecil Memeriksa apakah nilai di sebelah kiri lebih kecil daripada nilai di sebelah kanan. 5 < 3 false
    <= Lebih kecil atau sama dengan Memeriksa apakah nilai di sebelah kiri lebih kecil atau sama dengan nilai di sebelah kanan. 5 <= 3 false
