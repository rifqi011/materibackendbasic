Popup Javascript

  • Terdapat beberapa jenis yaitu alert, promt, dan confirm
  • Alert berupa informasi
  • Prompt berupa isian
  • Confirm berupa konfirmasi

Looping

  • Pengulangan sebuah instruksi sampai kondisi terpenuhi
  • Terdapat for loop, for in, for of, while, dan do while

For Loop

  • Menggunakan kata kunci for

For In

  • Menggunakan kata kunci for
  • Untuk me-looping sebuah object atau array

For Of

  • Menggunakan kata kunci for
  • Untuk me-looping sebuah object atau array
  • Namun harus dibungkus dengan Object.entries(object)

While

  • Versi yang lebih sederhana dibanding For loop
  • Melakukan pengulangan selama bernilai True

Do While

  • Varian dari While

Event

  • Event artinya aktivitas user
  • Misal klik, scroll

DOM

  • DOM merupakan Document Object Model
  • DOM dapat memanipulasi interface website

Mengambil element HTML

  • mengambil element berdasarkan id
  • mengambil element berdasarkan class
  • mengambil element berdasarkan nama tag
  • mengambil satu element berdasarkan selector
  • mengambil semua element berdasarkan selector
