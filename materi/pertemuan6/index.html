Javascript

Variable

  • Tempat menyimpan data
  • Keyword variable di JS: let, const, dan var

Tipe Data

  • Terdapat 2 jenis: Primitif dan Non-Primitif
  • Tipe Data Primitif: ...
  • Tipe Data Non-Primitif: ...

Tipe Data String

  • Mewakili data teks atau karakter
  • Diapit petik tunggal atau ganda ('/")

Tipe Data Number

  • Mewakili angka
  • Tanpa tanda petik
  • Bisa bulangan bulat maupun desimal

Tipe Data BigInt

  • Mewakili angka
  • Penulisannya sama seperti Number, tapi diakhiri huruf 'n'
  • Hanya bulangan bulat

Tipe Data Boolean

  • Mewakili benar dan salah
  • Penulisannya 'true' dan 'false' (tanpa tanda petik)

Tipe Data Undefined

  • Berarti tak terdefinisi
  • Ketika variable dideklarasikan tanpa inisialisasi atau tidak diberi value
  • Hanya berlaku bagi let dan var

Tipe Data Null

  • Berarti data kosong
  • Memberikan data kosong pada variable

Tipe Data Object

  • Menyimpan data dalam bentuk objek/JSON
  • Diapit kurung kurawal

Tipe Data Array

  • Menyimpan data dalam bentuk list
  • Diapit kurung siku
  • Dipisahkan dengan koma

Operator Aritmatika

  • Untuk Tipe Data Number
  • Mendukung: Penjumlahan(+), Pengurangan(-), Perkalian(*), Pembagian(/), Modulus(%)

Operator Logika

  • Untuk Tipe Data Bool
  • Mendukung: AND(&&), OR(||), NOT(!)
